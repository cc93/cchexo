#tag1 #tag2

标题和表格可用快捷键快速输入

  • 标题
  • 表格

快速打开文件（切换文件，显示最近文件） cmd + shift + O

内容检索

查找当前文件 cmd + F

全局查找，查找所有文件 cmd + shift + F

设置标签tag

虽然不支持为md文件设置标签tag属性，但是可用变通方法实现:

在文章内容中写入`#tag1 #tag2`，表示此文章的标签是tag1, tag2, 然后全局查找#tag1或#tag2就能找到此文件，从而实现标签功能
